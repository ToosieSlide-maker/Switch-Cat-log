Project instructions

- When giving code fixes or requested edits to the user, always use this format:
  `Busca esto:`
  exact code to find
  `Reemplaza por esto:`
  exact replacement code
- When the issue belongs to a function or self-contained block, provide the complete block from where it starts to where it closes, so there is no ambiguity.
- If the change is only 1 or 2 lines, provide only those exact lines in the same `Busca esto` / `Reemplaza por esto` format.
- Keep replacement instructions practical and ready to paste into GitHub or the local file.
